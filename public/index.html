TODO API HOME PAGE

======= BASIC ROUTES =========

Get All Available Todos
http Verb = GET
http://localhost:3000/v1/todos

For pagination, the server takes two query parameters ==== limit & page

Example: http://localhost:3000/v1/todos?=limit=5&page=1

Create new todo
http Verb = POST
http://localhost:3000/v1/todos

Update Todos Id
http Verb = UPDATE
http://localhost:3000/v1/todos/todoId

Delete todos by Id
http Verb = DELETE
http://localhost:3000/v1/todos/todoId
